# Background Images – AI Generation Guide

## What to do
1. Use Stable Diffusion, Leonardo.ai, or another free AI art platform.
2. Prompts: "Anime detective city, neon lights, rainy October, abandoned diner interior, cozy but mysterious, pixel-art style"
3. Generate or upscale to 1920x1080 for compatibility with hidden object and jigsaw puzzles.
4. Save as `diner_evening.png`, `clocktower_night.png`, etc. Drop them into this folder.

## Example Prompt
- "Anime pixel-art, moody detective office, rainy window, evidence board on wall, warm light, high detail"
