# Pixel Map – AI Generation Guide

How to create:
1. Use a pixel-art map generator (e.g., Dungeon Scrawl, or Stable Diffusion with prompts)
2. Prompts: "Pixel art, small town map, fall colors, neon lights, several districts, river docks, clock tower, orphanage, hospital, retro game HUD style"
3. Generate at least 1200x800px. Save as `fall_town_map.png` or similar.
4. Use as background for jigsaw and clue-finding puzzles.
